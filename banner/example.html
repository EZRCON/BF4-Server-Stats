Help with URL Query String Codes:

ServerID: (required) This is the server ID of the server you are trying to display. This is required.
ServerID=<number>

data: (optional) Without "Data=1" in the URL query string, the banner will be displayed as an image.
data=1

The following query stringes require "Data=1" to be in the URL query string:

bgcolor: (optional) This is the background color. This must be a 6-digit hexadecimal color.
bgcolor=<number>

fontcolor: (optional) This is the text font color. This must be a 6-digit hexadecimal color.
fontcolor=<number>

linkcolor: (optional) This is the link font color. This must be a 6-digit hexadecimal color.
linkcolor=<number>

sectionbgcolor: (optional) This is the background color of certain sections. This must be a 6-digit hexadecimal color.
sectionbgcolor=<number>

sectionfontcolor: (optional) This is the font color of certain sections. This must be a 6-digit hexadecimal color.
sectionfontcolor=<number>
