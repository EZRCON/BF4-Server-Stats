Code for displaying as image:

<img src="serverbanner.php?ServerID=3" alt="server" />


Code for displaying as default iframe:

<iframe src="serverbanner.php?ServerID=3&data=1" width="220px" height="712px" frameborder="0" scrolling="no"><p>iframes not supported</p></iframe>


Code for displaying as customized iframe:

<iframe src="serverbanner.php?ServerID=3&data=1&bgcolor=CCCCCC&fontcolor=000000&linkcolor=995555&sectionbgcolor=AAAAAA&sectionfontcolor=333333&onlinecount=20" width="220px" height="902px" frameborder="0" scrolling="no"><p>iframes not supported</p></iframe>


Help with URL Query String Codes:

ServerID: (required) This is the server ID of the server you are trying to display. This is required.
ServerID=<number>

data: (optional) Without "Data=1" in the URL query string, the banner will be displayed as an image.
data=1

The following query stringes require "Data=1" to be in the URL query string:

bgcolor: (optional) This is the background color. This must be a 6-digit hexadecimal color.
bgcolor=<number>

fontcolor: (optional) This is the text font color. This must be a 6-digit hexadecimal color.
fontcolor=<number>

linkcolor: (optional) This is the link font color. This must be a 6-digit hexadecimal color.
linkcolor=<number>

sectionbgcolor: (optional) This is the background color of certain sections. This must be a 6-digit hexadecimal color.
sectionbgcolor=<number>

sectionfontcolor: (optional) This is the font color of certain sections. This must be a 6-digit hexadecimal color.
sectionfontcolor=<number>

onlinecount: (optional) This is the number of players to show in the online players list. Fewer will take up less vertical space, more will take up more vertical space.
onlinecount=<number>

Hint: At the bottom of the iframe, it will give you the suggest iframe width and height values. Crop off the iframe to the suggested size to hide the suggested size information at the bottom.
